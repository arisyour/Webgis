# UTS WebGIS - Tema Kesehatan (Hijau-Biru Modern)
Petunjuk singkat:
1. Ekstrak folder `utsWebGis` ke folder server lokal Anda, misal:
   C:/xampp/htdocs/utsWebGis
2. Jalankan Apache & MySQL pada XAMPP.
3. Import file `init_db.sql` melalui phpMyAdmin.
4. Pastikan folder `uploads/` memiliki permission tulis.
5. Buka http://localhost/utsWebGis/login.php

Akun default:
- user: admin
- pass: admin123

Catatan:
- Klik pada peta untuk mengisi latitude dan longitude.
- Gambar disimpan di folder uploads/.
